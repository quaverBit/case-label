## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm install && npm run start

# Docker compose

Docker-compose:
# development mode
$ docker-compose up dev

# production mode
$ docker-compose up prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
